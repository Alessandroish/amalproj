# История чата — разработка сайта amalproj

Портфолио студента специальности **«Торговое дело»**. Хронология запросов
пользователя и того, что было сделано.

---

## 1. Старт — лендинг на Python
- Запрос: сайт на Python с parallax. Выбор: лендинг/портфолио на **FastAPI**.
- Создан базовый лендинг (Hero, Обо мне, Навыки, Проекты, Контакты) с parallax,
  кастомным курсором, анимациями.

## 2. Итерации по дизайну и анимациям
- Больше анимаций, кастомный анимированный курсор с хвостом из частиц.
- Минимализм; смена палитры (светлая → тёмная → снова светлая в белых тонах).
- Имя/название изменено на **amalproj**; зелёный скроллбар.
- Интерактивные буквы заголовка (увеличение у курсора, эффект только по строке).
- Индикатор прокрутки «Листайте вниз» (анимированный, по центру).

## 3. Функциональная система (бэкенд + админка)
- **FastAPI + SQLite**: весь контент в базе.
- **Админ-панель** `/admin` (вход логин + пароль + PIN): редактирование всего
  контента, проекты, навыки, приём заявок с формы контактов.
- Файлы: `main.py`, `db.py`, `content.py`, `templates/admin/*`, `static/css/admin.css`.
- Данные по умолчанию: `admin` / `admin123` / PIN `1234`.

## 4. Деплой-подготовка
- `Procfile`, `runtime.txt`, `requirements.txt`, `.gitignore`, `.env.example`, `README.md`.
- Git-репозиторий инициализирован, серия коммитов, ветка `main`.
- Объяснены GitHub и Railway; деплой — следующий шаг (домен позже).

## 5. Контент под торговое дело
- Hero «Развиваю розничную торговлю», навыки (закупки, мерчандайзинг,
  маркетинг, анализ рынка, ассортимент), кейсы (ассортиментная матрица,
  оптимизация закупок, маркетинговая акция), бегущая строка, статистика.

## 6. Заставки и переходы (много итераций)
- Печатающийся текст «Привет, я Амаль, учусь на специальности «Торговое дело»».
- Звук печатной машинки (Web Audio), регулировка скорости/мягкости.
- Переходы: панель → разворот-книга → полёт-портал 3D → раздвижные двери
  со свечением → линия-загрузка с процентами.
- **Итог:** заставка переделана в **ретро CRT-загрузку в стиле edh.dev**
  (зелёный фосфорный терминал, BOOTING…/STARTING…/READY., скан-линии,
  выключение монитора).

## 7. Эффекты и сторонние стили
- Liquid glass (матовое стекло iOS) на ключевых элементах.
- 3D-сцена в стиле **cosmos.so** (поле карточек), позже заменена на 3D-куб.
- Шапка в стиле **lunchbox.io** (плавающая пилюля, потом — терминальная панель).
- Parallax между секциями; анимированный график продаж; иконки + счётчики.

## 8. 3D-куб (Three.js)
- Кубик Рубика из `RoundedBox` + `MeshPhysicalMaterial`, RectAreaLights, anime.js.
- Сначала белый → затем **чёрный** глянцевый.
- Управление перетаскиванием мышью/тачем (без инверсии), инерция.
- Вынесен на **отдельную страницу `/press`** (ссылка «3D-куб» в меню).

## 9. Ретро тёмно-зелёная стилистика всего сайта
- Палитра: тёмный фон, фосфорно-зелёный текст/акценты, зелёные рамки.
- Моноширинный шрифт **JetBrains Mono** (с кириллицей).
- Глобальные CRT скан-линии + виньетка; терминальный вид компонентов.
- Бесшовный переход с ретро-заставки в ретро-сайт.

## 10. Переходы между секциями + поведение шапки
- **Стек-переход** секций: каждая накладывается поверх предыдущей при скролле
  (как колода карт / страницы книги) — через `position: sticky`.
- Шапка: убрана из «всегда fixed» — видна вверху и **появляется на последней
  секции** (Контакты), прячется между. *(в процессе доводки)*

---

## Технологии
- **Backend:** FastAPI, SQLAlchemy, SQLite
- **Frontend:** HTML/CSS/JS (vanilla), Three.js + anime.js (куб), Web Audio (звук)
- **Шрифт:** JetBrains Mono · **Тема:** ретро CRT тёмно-зелёный

## Структура проекта
```
main.py            — FastAPI: сайт + админка + заявки + /press
db.py              — SQLite (контент, проекты, навыки, заявки)
content.py         — схема и значения по умолчанию
templates/         — index.html, press.html, admin/*
static/css/        — style.css, admin.css
static/js/         — parallax.js (заставка, анимации), cube.js (3D-куб)
Procfile, runtime.txt, requirements.txt, README.md, .env.example
```
